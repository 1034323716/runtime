### Koupleless runtime 主要职责
1. Arklet： 模块运维管道，模块健康检查，模块 metric 采集
2. 多应用运行环境
3. 多应用治理动态 adapter
4. 多应用测试框架
5. 模块脚手架
6. 模块与基座通信注解与工具类
